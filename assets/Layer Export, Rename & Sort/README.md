# Layer Export, Rename & Sort

A UXP panel for Photoshop that combines layer export, renaming, and sorting tools in one place.

## Current Features

- Loads as a dockable non-blocking Photoshop panel.
- Exports selected layers with per-format settings for JPG, PNG, WebP, PSD, TIFF, and BMP.
- Renames selected layers with sequence, prefix/suffix, and remove/replace tools.
- Sorts layers by name with scope and order controls.
- Reads the current Photoshop layer selection live while the panel is open.
- Shows previews for export and rename actions.

## How to load it

1. Open `Adobe UXP Developer Tools`.
2. Choose `Add Plugin`.
3. Point it to this folder:
   `D:\MULTIMEDIA\_Tools & Supports\Actions & Scripts\_MAX_Photoshop Actions\Layer Export, Rename & Sort`
4. Launch the plugin into Photoshop.
5. Open the panel from the `Plugins` menu in Photoshop.
